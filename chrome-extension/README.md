### Phase 1: Load Extension
1. Open Chrome and navigate to `chrome://extensions`.
2. Enable **Developer Mode** (top right toggle).
3. Click **Load Unpacked**.
4. Select the `chrome-extension` folder inside this project.
5. **COPY THE EXTENSION ID** generated by Chrome (starts with `aaaaaaaa...`).

### Phase 1.5: Install Native Messaging Host
1. Open `chrome-extension/native-messaging/host-manifest.json` in your editor.
2. Replace `"chrome-extension://YOUR_EXTENSION_ID/"` with your **REAL Extension ID**.
3. Run the installer:
   ```bash
   cd chrome-extension/native-messaging
   ./install_native_host.sh
   # It will create the necessary manifest in ~/Library/Application Support/...
   ```

### Phase 2: Verify Connection
1. Click the Extension Icon in Chrome.
2. The popup should appear.
3. If successful, you will see a list of jobs from `Desktop/Google Auto`.
4. If you see an error, ensure the host manifest ID matches the extension ID.
